Modules
=======

This section provides detailed documentation for the various modules that make up the PySide Demo application.

.. toctree::
   :maxdepth: 2

   gui
   db
   theme
   main

The PySide Demo application is composed of three main modules:

1. GUI Module: Handles the graphical user interface components.
2. Database Module: Manages database operations and synchronization.
3. Main Module: Serves as the entry point for the application.

For detailed information about each module, please refer to their respective pages linked above.
